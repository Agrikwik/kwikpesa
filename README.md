# KwikPesa React Project

A modern, responsive React application for KwikPesa digital payments.

## Features

- **Modern React (Vite)**: High-performance development and build environment.
- **Edge-to-Edge Design**: Fluid full-width layout optimized for all screen sizes.
- **Bootstrap 5**: Styled with professional purple themes and Bootstrap icons.
- **Dynamic Routing**: Smooth client-side navigation via React Router.

## Getting Started

To run this project on your local machine, follow these steps:

### 1. Prerequisites
Ensure you have **Node.js (v18 or newer)** installed.

### 2. Installation
Clone the repository and install the dependencies:
```bash
git clone https://github.com/Agrikwik/kwikpesa.git
cd kwikpesa
npm install
```

### 3. Running Development Server
Start the local development server:
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser.

### 4. Direct Production Build
To build the project for production:
```bash
npm run build
```

## Project Structure

- `src/components`: Reusable UI components (Header, Footer).
- `src/pages`: Main page views (Home, Pricing, Contact, Login, Register).
- `src/App.jsx`: Main routing logic and layout.
- `public/assets/images`: Static image assets.
- `src/index.css`: Global styles including custom themes and responsiveness fixes.
